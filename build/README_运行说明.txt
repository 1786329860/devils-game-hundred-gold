《恶魔游戏：百金争夺》1.1.1

双击 DevilsGame.exe 即可运行。
请勿把 DevilsGame.pck 与可执行文件分开。

系统要求：
- Windows 10/11 64 位
- 支持 OpenGL 3.3 的显卡
- 建议分辨率 1280×720 或更高
- 联机模式需要可访问互联网（wss://tucao.aixiaolv.icu）
- 1.1.1 修复联机胜负回合掉线，并加入 WebSocket 心跳。

首次运行会在 Godot 用户数据目录创建 save.json，用于保存设置、成长、联机身份令牌和本地战绩。

本目录中的 GODOT_LICENSE.txt 是随程序分发的 Godot Engine 授权文本。
